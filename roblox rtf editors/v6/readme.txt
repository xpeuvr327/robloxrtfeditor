Roblox RTF Editor v6

Ce programme vous permet de créer du texte riche au format HTML en utilisant les fonctionnalités de Roblox, mais plus simple. Vous pouvez saisir du texte dans la boîte d'entrée, choisir des styles (gras, italique, souligné, barré) avec des cases à cocher, changer la couleur du texte avec des curseurs RGB ou des boîtes de saisie, et copier le texte transformé dans le presse-papier avec un bouton. Vous pouvez aussi voir la couleur choisie dans un carré de prévisualisation.

Pour utiliser ce programme, vous devez créer un dossier nommé v6, et y placer les fichiers index.html, script.js et readme.txt. Ensuite, vous devez ouvrir le fichier index.html avec votre navigateur web préféré. Vous pouvez alors commencer à créer du texte riche.

Les changements par rapport à la version précédente sont les suivants:

- Les curseurs RGB fonctionnent à nouveau
- Les curseurs RGB ont la couleur correspondante
- Les styles peuvent être combinés
- Le titre et les labels ont une police différente
- La boîte de sortie a la même largeur que la fenêtre
